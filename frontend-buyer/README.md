# Buyer Frontend - Optical Marketplace

Next.js 14+ frontend application for buyers.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- ✅ User Registration
- ✅ User Login
- ✅ Password Reset
- ✅ Authentication Context
- ✅ Protected Routes

## Tech Stack

- Next.js 14+
- React 19
- TypeScript
- Tailwind CSS
