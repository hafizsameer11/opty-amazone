# Admin Panel - Optical Marketplace

Next.js 16+ admin panel application with glassmorphism design.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3002](http://localhost:3002)

## Features

- ✅ Admin Authentication
- ✅ Dashboard with Statistics
- ✅ User Management
- ✅ Seller/Store Management
- ✅ Product Management (Approve/Reject)
- ✅ Order Management
- ✅ Category Management
- ✅ Coupons View
- ✅ Analytics
- ✅ Settings
- ✅ Activity Logs

- Next.js 16+
